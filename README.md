# Supermarket Shopping List 🛒

A simple React app to manage a list of products to buy from the supermarket.

## Features ✨
- Add a product
- Increase/decrease product amount
- Delete a product
- Beautiful and clean UI

## Technologies Used 🛠️
- React.js
- HTML5
- CSS3
